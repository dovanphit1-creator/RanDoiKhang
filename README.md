# 🐍 RanDoiKhang - Siêu Cấp Rắn Săn Mồi AI

Chào mừng bạn đến với **Trang Chủ** của RanDoiKhang! Đây là phiên bản Rắn săn mồi đối kháng thế hệ mới, tích hợp trí tuệ nhân tạo (AI) có khả năng tự học và phát triển.

---

## 🚀 CHỌN PHIÊN BẢN CỦA BẠN

### 🌐 1. Chơi Ngay (Dành cho iPhone/iPad/Android/Web)
Không cần tải về, nhấn vào link bên dưới để chơi trực tiếp trên trình duyệt:
> **[👉 NHẤN VÀO ĐÂY ĐỂ CHƠI NGAY](https://dovanphit1-creator.github.io/RanDoiKhang/)**
*(Lưu ý: Trên iPhone, hãy chọn "Thêm vào màn hình chính" để có trải nghiệm tốt nhất)*

### 💻 2. Tải về cho Máy tính (Windows)
1. Nhấn vào tab **[Actions](https://github.com/dovanphit1-creator/RanDoiKhang/actions)** ở phía trên.
2. Chọn tiến trình "Build Windows EXE" gần nhất.
3. Kéo xuống mục **Artifacts** để tải file `.exe`.

### 📱 3. Tải về cho Android (.APK)
1. Nhấn vào tab **Actions**.
2. Chọn tiến trình "Build APK" gần nhất.
3. Tải file trong mục **Artifacts**.
